# 🚀 Meu Portfólio

Este é o meu portfólio pessoal, desenvolvido com tecnologias modernas para criar uma experiência web rápida, atraente e totalmente responsiva.

## 🛠️ Tecnologias Utilizadas

- **React 18** - Biblioteca principal para a interface.
- **Vite** - Build tool extremamente rápida.
- **TypeScript** - Tipagem estática para maior segurança e produtividade.
- **Tailwind CSS** - Estilização baseada em utilitários.
- **Framer Motion** - Animações fluidas e interações avançadas.
- **shadcn/ui** - Componentes de interface acessíveis e elegantes.
- **Lenis** - Scroll suave de alta performance.
- **Lucide React** - Conjunto de ícones consistentes.

## ✨ Funcionalidades

- 📱 **Design Responsivo**: Adaptado para qualquer tamanho de tela (Mobile, Tablet e Desktop).
- 🌓 **Suporte a Temas**: Alternância entre modo claro e escuro.
- ⚡ **Performance Otimizada**: Carregamento rápido graças ao Vite.
- 🎨 **Animações Modernas**: Efeitos de transição e scroll suaves.
- 🌐 **Internacionalização**: Estrutura preparada para múltiplos idiomas (Context API).

## 🚀 Como Executar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/doglz/my-portfolio.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd meu-portfolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   # ou
   bun install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   # ou
   bun dev
   ```

5. **Acesse no navegador:**
   `http://localhost:5173`
